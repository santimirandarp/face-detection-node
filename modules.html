faceDetection

  • faceDetection(configPaths: ConfigPaths, configModel?: Partial<ConfigModel>): Promise<void>
    • Defined in src/faceDetection.ts:67
    • Recursively grabs all images starting at "imgDir",
    • runs face detection on them
    • saves the results to "outDirPath".

    It will also prompt you to confirm the paths (just once) You can pass confirmPaths:false in the configPaths to remove this.

    example
    const configPath = {
    // this is your absolute path to "your script" folder
    absPathToDir: __dirname,
    // assumes you have a folder called `images` in the same folder as this file
    // in the special case of `singleFile:true`, you can pass a file path instead
    imgDir: "./images",
    // assumes you have a folder called `models` in the same folder as this file
    modelsDir: "./models",
    // writes output to file `stats.json` in the same folder as this file
    outStatsPath: "./stats.json",
    };

    const configModel = {
    singleFace: false,
    // with this option set to true, 
    // you can also give imgDir a file path
    // instead of a directory
    singleFile: true,
    detectionOptions: new SsdMobilenetv1Options({
    minConfidence: 0.36,
    maxResults: 2,
    }),
    };
    // run the detection
    faceDetection(configPath, configModel);
